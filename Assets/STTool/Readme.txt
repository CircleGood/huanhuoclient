  [Summeray]
    STTool is the abbreviation of "Script Template Tool", this tool can
    be used to edit Unity3D engine's script template and aoto add file
    header comment when creat script file in Unity3D editor.
--------------------------------------------------------------------------
  [Environment]
    This tool applies to Unity3D 5.0, .Net Framework 3.0 or above version.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/STEditor" in Unity3D editor's menu bar and
    click it to open the STEditor window or press combination key Alt+S.
    Select "EditTarget"(example CSharp Behaviour), click the "Currnet"
    button to browse current script template.
    Open the corresponding template file(example CsharpBehaviour.txt) and
    copy the text to "ScriptTempalte" text edit area of STEditor, and
    click the "Save" button.
    STTool will be work when a script file(example C# Script) is created.
--------------------------------------------------------------------------
  [Template]
    The templates(example CsharpBehaviour.txt) store in the path 
    "Assets/STTool/Template". in fact, you can use the [Mark] to  create
    your own style templates.
--------------------------------------------------------------------------
  [Mark]
    #SCRIPTNAME#, #CopyrightTime#, #CreateTime#.
